# Fuel Efficiency Prediction 🚗⛽

## Project Overview
Fuel Efficiency Prediction is a Machine Learning based project that predicts the fuel efficiency (MPG - Miles Per Gallon) of a vehicle using different vehicle parameters.

The project uses a Deep Learning model built with TensorFlow/Keras and provides an interactive web interface using Streamlit.

## Features
- Predict vehicle fuel efficiency using input features
- User-friendly Streamlit web application
- Trained Neural Network model using TensorFlow
- Data preprocessing and model evaluation
- Fast and real-time predictions

## Dataset
Dataset used: Auto MPG Dataset

The dataset contains information about vehicles such as:
- Cylinders
- Displacement
- Horsepower
- Weight
- Acceleration
- Model Year
- Origin

Target Variable:
- MPG (Miles Per Gallon)
